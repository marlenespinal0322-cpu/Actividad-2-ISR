¿Qué hace el programa?
Calcula el Impuesto Sobre la Renta (ISR) de un empleado.

Datos de entrada:
- Sueldo mensual

Datos que procesa:
Calcula el ISR según la escala correspondiente.

Datos que imprime:
- Sueldo
- ISR o el mensaje "No aplica para descuento de ISR".
